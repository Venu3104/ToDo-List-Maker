About This Project

My-ToDo is a simple web-based ToDo list application built with Flask and Bootstrap. It allows users to add, update, delete, and search their tasks efficiently.

How to Use

  1. To add a new task, fill in the "ToDo List" and "ToDo Description" fields on the home page and click Submit.
  2. Your tasks will appear in the "MY TODO List" table below the form.
  3. To update a task, click the Update button next to it, edit the details, and submit.
  4. To delete a task, click the Delete button next to it.
  5. Use the Search bar in the navigation to filter your tasks in real-time.

Development

  • Backend: Python, Flask, Flask-SQLAlchemy (SQLite database)
  • Frontend: HTML, Bootstrap 5, JavaScript (for live search)
  • Features: Add, update, delete, and search tasks; responsive design; persistent storage.
  • Developer: Venu

This project demonstrates the basics of full-stack web development using Flask and Bootstrap, including CRUD operations and AJAX-based search.
